# Lô Tô Online

## 📌 Về Dự Án
**Lô Tô Online** là phiên bản số hóa tiện lợi của trò chơi dân gian Lô Tô quen thuộc, mang lại trải nghiệm giải trí thú vị ngay trên trình duyệt web. Việc số hóa giúp người chơi dễ dàng tham gia cùng bạn bè, gia đình mọi lúc mọi nơi mà không cần đến bộ cờ truyền thống.

## 🚀 Tính Năng Chính
- **Quản Lý Lô Tô:** Quay số ngẫu nhiên, tự động hiển thị số đã kêu.
- **Giao Diện Dò Số Trực Quan:** Bảng dò dễ nhìn, tích hợp khả năng đối chiếu kết quả theo thời gian thực.
- **Tối Ưu Hóa Đa Nền Tảng:** Thiết kế thân thiện cả trên máy tính (Desktop) lẫn điện thoại (Mobile).

## 🛠️ Công Nghệ Sử Dụng
- **Công Nghệ Web:** HTML, CSS, JavaScript (Vanilla JS hoặc công nghệ web hiện đại)
- **Hosting / Deployment:** Đóng gói và triển khai trực tiếp trên Github Pages, đảm bảo tải trang cực nhanh chóng.
